# v0.6.0-rc.8

This release candidate fixes the live acceptance harness after RC.7 proved
bootstrap provisioning, immutable artifact binding, cache initialization, and
exact cloud discovery but stopped before RKE2 installation.

The verified release-image manifest is an Ansible host fact owned by
`localhost`. The bastion play now imports that exact in-memory fact explicitly
through `hostvars` before proving every cached controller digest. It does not
reread the manifest from disk or weaken the existing immutable-artifact
boundary.

Regression coverage parses the playbook and requires the explicit cross-host
handoff. RC.7 automatic recovery deleted its bastion, three control-plane VMs,
four floating IPs, and both firewalls; three strict account-wide reads then
matched the pre-mutation baseline with zero differences across every audited
resource kind.
